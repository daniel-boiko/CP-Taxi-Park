---Идеи для осуществления синка между водителем и автомобилем---
    У автомобиля есть индивидуальный номер
    Можно сначала вводить автомобили, а при вводе водителей указывать номер автомобиля для синка


    ARRAY_MERGE!!!
    с readline можно синкать


    Считываем ячейку массива через readline []
    Далее присваиваем ячейку к переменной и делаем из неё массив
    Рабочий пример:

    $listDriver[1] = array (0 => 'driver');
    $listCar[1]= array(0 => 'car');

    $result = array_merge($listCar[1], $listDriver[1]);
    print_r ($result);
